Feature Navigator - Help

Overview

Feature Navigator is a QGIS 3.28.8 plugin designed to help users quickly navigate through features in a selected vector layer.

Main Features

  • Navigate forwards and backwards through features
  • Automatically zoom to selected features
  • Autoplay mode for sequential navigation
  • Optional spacebar play/pause shortcut
  • Automatically updates when features are added or removed

How To Use

  1. Open the Feature Navigator plugin from the QGIS toolbar or Plugins menu.
  2. Select a vector layer using the layer dropdown.
  3. Use the Next and Previous buttons to browse features.
  4. Press Play to automatically move through features.
  5. Press Pause to stop autoplay.
  6. Press the help button in the settings dialog to open this help page.

Settings

The settings dialog allows you to configure:

  • Playback Speed - Time interval between feature changes during autoplay
  • Zoom Buffer - Additional map extent around selected features
  • Spacebar Play/Pause - Enable or disable the spacebar shortcut

Help Button

The help button opens the Feature Navigator documentation directly from within the settings dialog.

This provides quick access to:

  • Plugin usage instructions
  • Navigation controls
  • Autoplay settings
  • Feature behaviour notes

Notes

  • Only vector layers with geometry are supported.
  • The plugin automatically makes hidden layers visible when needed.
  • Feature navigation is based on QGIS feature IDs.
