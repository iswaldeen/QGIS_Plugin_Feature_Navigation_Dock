Feature Navigation Dock - Help

Overview

Feature Navigation Dock is a QGIS plugin that allows you to quickly move through features in a vector layer.

The plugin supports manual navigation, autoplay navigation, and navigation through selected features only.

Main Features

  • Navigate forwards and backwards through features
  • Automatically zoom to features
  • Autoplay feature navigation
  • Pause and resume autoplay
  • Optional spacebar play/pause shortcut
  • Optional looping navigation mode
  • Optional navigation slider bar
  • Quick slider-based feature navigation
  • Live slider position updates during navigation and autoplay
  • Navigate only selected features
  • Automatically refresh when selections or features change
  • Automatically turns on hidden layers during navigation

How To Use

  1. Open Feature Navigation Dock from the QGIS toolbar or Plugins menu.
  2. Select a vector layer using the layer dropdown.
  3. Use Next and Previous to move through features.
  4. If the navigation slider is enabled in settings, drag the slider to quickly move through features.
  5. Press Play to start autoplay navigation.
  6. Press Pause to pause autoplay.
  7. To navigate only selected features:
    • Select features in QGIS
    • Open the settings dialog
    • Enable Navigate through selected features only
  8. If the selected layer is hidden, the plugin will turn it on automatically when you press Next or Play.
  9. Open Plugins → Feature Navigation Dock → Settings to change plugin settings.
  10. Open Plugins → Feature Navigation Dock → Help to open this help page.

Settings

  • Playback Speed - Time between autoplay feature changes
  • Zoom Buffer - Extra map space around selected features
  • Spacebar Play/Pause - Enable or disable the spacebar shortcut
  • Navigate through selected features only - Restrict navigation to selected features
  • Loop Navigation - Restart navigation from the beginning when the final feature is reached
  • Show Navigation Slider - Show or hide the feature navigation slider bar

Autoplay Behaviour

  • Autoplay moves through features automatically
  • The navigation slider updates automatically during autoplay
  • Pause keeps the current playback countdown
  • Play resumes from the remaining countdown time
  • Changing playback speed resets the countdown using the new timer value
  • If looping navigation is disabled, autoplay stops when the final feature is reached

Selected Feature Navigation

When selected feature mode is enabled, the plugin only navigates through currently selected features.

Selections can be made:

  • In the map canvas
  • From the attribute table
  • Using QGIS selection tools

The feature list automatically refreshes when selections change.

Notes

  • Only vector layers with geometry are supported
  • If a selected layer is hidden, the plugin turns it on and shows a message to let you know.
  • Navigation is based on QGIS feature IDs
  • The navigation slider provides a visual indicator of your current position in the layer
  • Selected feature mode requires selected features in the active layer
  • Plugin menu icons automatically adjust for light and dark QGIS themes
